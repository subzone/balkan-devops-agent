About

Balkan DevOps Agents brings personality and expertise to your VS Code workflow. Each agent is a specialized DevOps professional with their own unique character, communication style, and area of expertise. From garbage collection to cloud cost optimization, from security audits to architecture reviews—these agents speak your language (literally, in Serbian!) and get the job done.

Installation

From VS Code Marketplace

ext install subzone.balkan-devops-agents

Or via VS Code UI

  1. Open VS Code
  2. Press Cmd+Shift+X (macOS) or Ctrl+Shift+X (Windows/Linux)
  3. Search for "Balkan DevOps Agents"
  4. Click Install
  5. Reload VS Code
  6. Open Copilot Chat and type @steva to test!

From VSIX (Manual)

code --install-extension balkan-devops-agents-1.0.0.vsix

Requirements

  • VS Code version 1.116.0 or higher
  • GitHub Copilot extension installed and active
  • GitHub Copilot Chat enabled
  • Active GitHub Copilot subscription
