O Ekstenziji

Balkan DevOps Agents donosi osobenost i stručnost u vaš VS Code workflow. Svaki agent je specijalizovani DevOps profesionalac sa svojim jedinstvenim karakterom, stilom komunikacije i oblašću ekspertize. Od garbage collection-a do cloud cost optimizacije, od security audita do architecture review-a—ovi agenti pričaju srpski i obavljaju posao!

Instalacija

Sa VS Code Marketplace-a

ext install subzone.balkan-devops-agents

Ili preko VS Code korisničkog interfejsa

  1. Otvori VS Code
  2. Pritisni Cmd+Shift+X (macOS) ili Ctrl+Shift+X (Windows/Linux)
  3. Pretraži "Balkan DevOps Agents"
  4. Klikni na Install
  5. Restartuj VS Code
  6. Otvori Copilot Chat i kucaj @steva da testiraš!

Sa VSIX fajla (Ručno)

code --install-extension balkan-devops-agents-0.0.2.vsix

Zahtevi

  • VS Code verzija 1.116.0 ili viša
  • GitHub Copilot ekstenzija instalirana i aktivna
  • GitHub Copilot Chat omogućen
  • Aktivna GitHub Copilot pretplata
